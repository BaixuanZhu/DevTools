# UI 样式细节优化实施计划

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** 重构布局为 Sidebar 全高独立面板，优化 Sidebar/Footer/Focus 样式细节。

**Architecture:** Sidebar 改为 `position: fixed` 占满左列，内部分层（顶部 Logo sticky + 下方导航可滚动）；右侧区域 `ml-[240px]` 包含 sticky Header + 可滚动 Content + Footer；仅输入框保留 focus 样式。

**Tech Stack:** Astro 6, Vue 3, Tailwind CSS v4, Alpine.js
